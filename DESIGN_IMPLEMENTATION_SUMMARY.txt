1. Always use Apple HIG hex codes:
   ✅ bg-[#2C2C2E]    (NOT bg-gray-800)
   ✅ text-[#98989D]  (NOT text-gray-400)
   ✅ border-[#38383A] (NOT border-gray-700)

2. Maintain consistent transitions:
   ✅ transition-colors duration-150

3. Add focus states for accessibility:
   ✅ focus:outline-none focus:ring-2 focus:ring-[#0A84FF]

4. Use proper text hierarchy:
   ✅ text-white → text-[#98989D] → text-[#636366]

5. Include hover states:
   ✅ hover:bg-[#3A3A3C] hover:border-[#48484A]
