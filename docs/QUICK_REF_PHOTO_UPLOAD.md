# 📸 QUICK REFERENCE: Photo Upload (New UX)

## ✨ New Workflow

```
1. Enter Title (required *)
   ├─ Type manually OR
   └─ Click "Auto" button

2. Select Photos
   └─ Click "Choose Photos"

3. Review Preview
   ├─ See file names
   ├─ See file sizes
   └─ Click "Clear All" if needed

4. Upload
   └─ Click "Upload Photos" button
```

## ⚠️ Validation

| ❌ Cannot Upload If: |
|---------------------|
| Title is empty |
| No files selected |
| Already uploading |

| ✅ Can Upload When: |
|---------------------|
| Title filled in |
| Files selected |
| Not uploading |
